Installation
------------
To install from pip:

    pip install datadog

## Using
Command line usage:
``` bash
export DATADOG_API_KEY=XXXXXXXX
export DATADOG_APP_KEY=YYYYYYYY
python ./ss.py
```
Then on any browser go to:

http://localhost:8080
